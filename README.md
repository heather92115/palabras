# Simple, free and open source word and phrase pairs to learn a new language.

## Prerequisites 
1. Rust
2. Cargo
3. Docker(optional)
4. Postgres database

This project is created with Rust and Cargo.
To build the project:
> cargo build

To update the project:
> cargo update
